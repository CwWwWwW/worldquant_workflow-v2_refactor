# Security Notes

See also the root [SECURITY.md](../SECURITY.md).

Do not commit:

- secrets or credentials
- `config.json`
- `.env` or `.env.*`
- cookies or browser session storage
- browser profiles
- `runtime/db`
- screenshots
- platform private data
- account usernames/passwords
- private Alpha results
- production logs with sensitive content

Security reports should avoid real credentials, private platform data, and sensitive screenshots. If private GitHub security reporting is available, use it; otherwise contact the maintainer through GitHub.

This project does not accept real credential samples or private platform data in public issues, pull requests, or discussions.
