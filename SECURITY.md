# Security Policy

## Sensitive Data

Never commit:

- `config.json`
- `.env` or `.env.*`
- credentials
- API keys
- cookies
- browser profiles or browser storage
- `runtime/db`
- logs
- screenshots
- private alpha results
- platform-owned confidential data

## Reporting

If a security issue is found, open a private report if available or contact the maintainer through GitHub.

Do not include real credentials, private platform data, or screenshots with sensitive information in public issues.

This project does not accept real credential samples, cookies, browser storage, private datasets, private Alpha results, or platform-owned data in issues, pull requests, or discussions.

## Supported Versions

Only the latest `main` branch and latest GitHub Release are considered supported.
